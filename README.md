# 🏡 Property Data Analysis for Property-GPT

An interactive real estate data analysis dashboard built with **Python**, **Streamlit**, **Pandas**, and **Plotly**.

This project provides powerful tools for exploring and analyzing real estate sales and rental data. Users can filter datasets, visualize market trends, compare property prices, and gain valuable insights through interactive dashboards.

---

## 📌 Features

- 📊 Interactive Streamlit dashboard
- 🏠 Property sales and rental analysis
- 📈 Interactive Plotly visualizations
- 📉 Price distribution analysis
- 📍 Neighborhood-based filtering
- 🏢 Property type comparison
- 📐 Price per square meter analysis
- 📋 Statistical summaries
- 🎨 Responsive and user-friendly interface

---

## 🚀 Installation

Clone the repository

```bash
git clone https://github.com/parsariazi/property-data-analysis-for-property-GPT.git
```

Move to the project directory

```bash
cd property-data-analysis-for-property-GPT
```

Install the required packages

```bash
pip install -r requirements.txt
```

---

## ▶️ Run the Application

Start the Streamlit application:

```bash
streamlit run src/property_analytics.py
```

The application will be available at:

```text
http://localhost:8501
```
